Setup on New Machine
Extract the files
Install Python 3.13+, Node.js, and ffmpeg
Run these commands:

cd ytdlp
python3 -m venv venv
source venv/bin/activate
pip install uv
uv pip install -e .
cd ui && npm install && npm run build && cd ..
./start.sh
